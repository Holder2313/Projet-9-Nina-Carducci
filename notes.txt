<PERFORMANCES>
---------------


Optimisation des images :  redimensionner et réduire la taille, pour accelerer le chargement.

Réduction du JavaScript et CSS bloquant le rendu : Éliminer ou différer le chargement des ressources qui bloquent le rendu


Minification des ressources : Réduire la taille des fichiers CSS et JavaScript en les minifiant, ce qui réduit le temps de téléchargement.

Utilisation de la mise en cache efficace : Configurer les en-têtes de cache pour les ressources statiques afin de réduire les temps de chargement pour les visiteurs récurrents.

Réduction du déplacement de mise en page (Cumulative Layout Shift) : Stabiliser les éléments visuels pendant le chargement pour améliorer l'expérience utilisateur.
